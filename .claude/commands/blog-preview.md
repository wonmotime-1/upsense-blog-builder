description: 작성된 글 폴더의 발행 어시스턴트(미리보기 + 복사 버튼) 열기
argument-hint: <폴더명 또는 경로>
---

"$ARGUMENTS" 폴더의 글에 대해 발행 어시스턴트를 엽니다.

1. 폴더 경로 추정:
   - `2026-04-09_상세페이지비용` 같은 폴더명만 줬다면 `output/$ARGUMENTS`
   - 전체 경로면 그대로 사용

2. preview.js 실행:
   ```bash
   node scripts/preview.js --folder "output/..."
   ```

3. 자동으로 브라우저가 열리며 다음 기능 제공:
   - 제목·태그·메타설명 복사 버튼
   - 본문 섹션별 "서식 포함 복사" / "텍스트만 복사" 두 모드
   - 이미지 4장 개별/일괄 다운로드
   - 발행 체크리스트 (10개 항목)

4. 사용자에게 안내:
   "브라우저가 열렸습니다. 위에서부터 차례로 복사해서 스마트에디터에 붙여넣으시면 됩니다. 표는 복사가 안 되니 스마트에디터에서 직접 만드세요."
